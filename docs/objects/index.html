Objects

An object is a central entity of o42a.

Object has:

  • a value of some basic type,
  • a definition - a value evaluation algorithm,
  • fields - a nested objects with identifiers,
  • adapters - an additional object interfaces,
  • an ancestor - a reference to a object this one inherited from.
